# Taller_Semana6
# Evaluador Automatizado de Siniestros - InsurTech

## Desafío Planteado:
Analizar reclamos de seguros de autos. Aprobar el pago automáticamente si el monto es < X y no hay historial de siniestros, o cambiar el estado a "Revisión Manual" si levanta banderas rojas.

## Pasos para completar el desafío

 
Este proyecto nació del análisis de pólizas de seguros del Ecuador. El primer paso fue crear un **Story Board** basándose en un documento de póliza de seguro de Ecuador, lo que permitió entender las reglas de negocio, los flujos y las necesidades reales de las aseguradoras.
 
A partir de ese análisis, se generaron los entregables del taller

##### 1. Story Board para análisis inicial
##### 2. PRD (Product Requirements Document)
##### 3. USER_STORIES (Historias de Usuario)
##### 4. SUBTASKS (Tasking y Estimación)
##### 5. Tablero de GitHub Projects
