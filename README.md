# Taller_Semana6
# Automatizador de Siniestros - InsurTech


## Contexto del Proyecto
 
Este proyecto nació del análisis de pólizas de seguros del Ecuador. El primer paso fue crear un **Story Board** basándose en un documento de póliza de seguro de Ecuador, lo que permitió entender las reglas de negocio, los flujos y las necesidades reales de las aseguradoras.
 
A partir de ese análisis, se generaron los entregables del taller

##### 1. PRD (Product Requirements Document)
##### 2. USER_STORIES (Historias de Usuario)
##### 3. SUBTASKS (Tasking y Estimación)
##### 4. Tablero de GitHub Projects
##### 5. Story Board oara análisis
